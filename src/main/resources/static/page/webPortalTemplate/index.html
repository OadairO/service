云杉苗种植基地 / 03

云杉幼苗对干燥的抵抗力弱，耐阴湿，应经常浇水以保持湿润，对阳光抵抗力弱，接草后应架设荫棚，以避免日灼危害。苗木生长缓慢，一般当年不进行间苗。云杉幼树期易受晚霜为害，故多设荫棚、栽植在高大苗木下方，冬季进行保护防御。云杉不易移植，多采取带土移植，移植时应仔细操作，减少对根系和枝叶的伤害，以提高成活率。...
